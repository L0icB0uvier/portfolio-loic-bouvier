## Description de l'outil

Easy Spawn est un outil permettant d'instancier rapidement des prefabs dans Unity afin d'accélérer le processur de level design. L'interface d'Easy Spawn est accessible à la fois dans une (ou plusieurs) fenêtre(s) éditeur dédié(s) ou dans directement dans la fenêtre Scene dans un Overlay. 

L'utilisateur peut librement assigner et organiser ses prefabs et customiser l'apparence des différentes fenêtres.
L'outil a été pensé pour fonctionner autant en 2D qu'en 3D et offre un grand nombre d'options sur la manière d'instancier les prefabs.

## Mon rôle sur le projet

J'ai entièrement codé l'outil pour l'un des projet sur lequel je travaillais. Ayant constaté son potentiel à fortement amélioré le workflow des level designers, j'ai décidé d'améliorer l'Asset pour le commercialiser.
