## Description du Projet

Ce projet a compris la réalisation de 2 mini-jeux pour le compte de Sanofi afin d'être présenté sur leur stand pendant le salon VivaTech 2024.

Les mini-jeux, à porté éducative, ont servi à promouvoir le développement de leurs outils d'intelligence artificielle.
Chaque jeu comprend une partie éducative permettant d'expliquer les problématiques et une partie jeu permettant d'illustrer ces dernières.

## Mon rôle sur le projet

En collaboration avec Snack Interactive, je me suis chargé de toute l'aspect technique du projet dans Unity.

Mes tâches principales ont consisté à:
- Mettre en place l'interface utilisateur sous forme de slides
- Programmer le gameplay des mini-jeux
- Intégrer les visuels en collaboration avec leurs graphistes
